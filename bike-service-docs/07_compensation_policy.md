# 赔偿标准与政策

## 文档类型
- **文档名**: `CompensationPolicy`
- **适用范围**: 全场景客服赔偿参考标准
- **更新日期**: 2026-03-27

## 概述

本文档规定了智能单车客服在各类投诉场景下的标准赔偿政策，客服人员须严格按照本标准执行，超出权限范围需上报主管审批。

---

## 一、客服赔偿权限分级

| 级别 | 执行人 | 赔偿上限 | 说明 |
|------|--------|---------|------|
| L1 | 普通客服 | 现金 ≤ 50元 / 骑行券不限 | 日常投诉处理 |
| L2 | 高级客服/组长 | 现金 51-200元 | 较复杂投诉 |
| L3 | 主管 | 现金 201-1000元 | 重大投诉/人身安全 |
| L4 | 运营总监/法务 | 现金 > 1000元 | 涉及法律纠纷 |

---

## 二、各场景赔偿标准

### 2.1 开锁失败

| 情况 | 赔偿内容 | 执行级别 |
|------|---------|---------|
| 设备故障导致无法开锁 | 1张5分钟免费骑行券 | L1 |
| 设备故障导致等待超10分钟 | 1张30分钟免费骑行券 | L1 |
| 多次开锁失败影响出行 | 1张30分钟骑行券 + 酌情1-5元红包 | L1-L2 |

### 2.2 计费争议

| 情况 | 赔偿内容 | 执行级别 |
|------|---------|---------|
| 系统多扣费 | 退还多扣金额 + 1张5元优惠券 | L1 |
| 重复扣费 | 退还重复金额 + 1张5元优惠券 | L1 |
| 优惠券/骑行卡未生效（系统bug） | 退还优惠金额 + 1张同等面值新券 | L1 |
| 未正确还车导致多扣费（系统责任） | 退还多扣金额 + 1张骑行券 | L1 |

### 2.3 车辆损坏

| 情况 | 赔偿内容 | 执行级别 |
|------|---------|---------|
| 用户上报安全隐患（刹车/车轮） | 1张30分钟骑行券（感谢上报） | L1 |
| 用户上报一般损坏 | 1张5分钟骑行券 | L1 |
| 骑行中遭遇损坏影响体验 | 1张30分钟骑行券 + 酌情减免本次费用 | L1 |
| 骑行中车辆损坏导致用户轻微受伤 | 报销医疗费（凭发票）+ 500元慰问金 | L3 |
| 骑行中车辆损坏导致用户严重受伤 | 启动保险理赔 + 法务介入 | L4 |

### 2.4 押金退款

| 情况 | 赔偿内容 | 执行级别 |
|------|---------|---------|
| 退款超时未到账（系统原因） | 补偿5元优惠券 | L1 |
| 退款金额错误（系统原因） | 补退差额 + 5元优惠券 | L1 |

### 2.5 账户问题

| 情况 | 赔偿内容 | 执行级别 |
|------|---------|---------|
| 系统错误导致账户被误封 | 解封 + 1张30分钟骑行券 | L1 |
| 账户被盗导致财产损失 | 退还被盗金额 + 5元优惠券 | L2 |
| 系统问题导致优惠券丢失 | 补发等额优惠券 | L1 |

### 2.6 违规停车申诉

| 情况 | 赔偿内容 | 执行级别 |
|------|---------|---------|
| GPS误判撤销罚款 | 退还罚款 + 1张5分钟骑行券 | L1 |
| 标识不清撤销罚款 | 退还罚款 + 1张5分钟骑行券 | L1 |
| 因违规记录导致账户被误封 | 解封 + 1张30分钟骑行券 | L1 |

## 三、骑行券说明

### 骑行券类型
| 券类型 | 面值 | 有效期 | 使用条件 |
|--------|------|--------|---------|
| 5分钟免费骑行券 | 免费骑行5分钟 | 30天 | 无门槛 |
| 30分钟免费骑行券 | 免费骑行30分钟 | 30天 | 无门槛 |
| 5元优惠券 | 抵扣5元 | 30天 | 单次消费满1元可用 |
| 10元优惠券 | 抵扣10元 | 30天 | 单次消费满5元可用 |

### 骑行券发放方式
- 客服系统直接发放到用户账户
- 发放后用户在 App 内"我的" → "优惠券"中查看
- 发放后立即生效

## 四、现金退款流程

### 退款发起
1. 在客服系统中创建退款工单
2. 填写退款金额、原因、用户信息
3. L1权限内直接提交；超出权限需上级审批

### 退款时效
| 支付方式 | 到账时效 |
|---------|---------|
| 微信支付 | 1-3 个工作日 |
| 支付宝 | 1-3 个工作日 |
| 银行卡 | 3-7 个工作日 |
| 平台余额 | 实时到账 |

### 退款凭证
- 退款发起后，系统自动生成退款流水号
- 告知用户流水号，用于查询退款进度
- 退款完成后 App 内推送通知

---

## 五、人身安全事故处理流程

### 轻微受伤（擦伤、轻微扭伤）
1. 第一时间关心用户，建议就医
2. 告知用户保留就医凭证（挂号单、诊断书、发票）
3. 上报主管，启动 L3 赔偿流程
4. 报销合理医疗费用（上限500元，凭发票）
5. 额外补偿500元慰问金

### 严重受伤（骨折、需住院）
1. 第一时间关心用户，确认已就医
2. 立即上报主管和法务部门
3. 启动意外险理赔流程
4. 法务介入，按保险条款和法律规定处理
5. 全程跟进，保持与用户沟通

### 注意事项
- 客服人员不得擅自承诺赔偿金额
- 所有人身安全事故必须上报，不得私自处理
- 保留所有沟通记录作为证据

---

## 六、不予赔偿的情形

以下情形经核实后不予赔偿：
1. 用户违规使用（超速、逆行、载人等）导致的损失
2. 用户故意损坏车辆
3. 用户提供虚假信息骗取赔偿
4. 因用户自身原因（如未看路）导致的事故
5. 不可抗力因素（自然灾害等）

---

## 七、赔偿记录与统计

### 记录要求
- 每笔赔偿必须在客服系统中记录
- 记录内容：用户ID、投诉类型、赔偿内容、处理人、时间
- 超权限赔偿需附审批记录

### 异常赔偿上报
- 单用户单月赔偿超过3次，需上报组长关注
- 疑似恶意投诉骗取赔偿，上报风控团队
