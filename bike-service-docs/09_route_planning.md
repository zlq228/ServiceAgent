# 骑行路线规划服务

## 问题类型
- **问题名**: `RouteQuery`
- **问题级别**: 中优先级
- **触发条件**: 用户询问骑行路线、距离或时间

## 问题描述
用户需要了解：
- 两地之间的骑行距离
- 预计骑行时间
- 推荐的骑行路线
- 如何避开拥堵路段

## 可用工具
- `calculate_distance`: 计算两地直线距离
- `plan_route`: 规划详细骑行路线
- `search_location`: 搜索起点和终点位置

## 路线规划步骤

### 步骤1: 确认起点和终点
使用 `search_location` 工具分别搜索起点和终点的位置信息

### 步骤2: 计算距离
使用 `calculate_distance` 工具获取两地距离

### 步骤3: 规划路线
使用 `plan_route` 工具获取详细路线和预计时间

### 步骤4: 提供建议
根据路线信息，提供骑行建议和注意事项

## 热门路线推荐

### 商务区通勤路线
**国贸 → 三里屯**:
- 距离: 约 3.5-4 公里
- 预计时间: 15-20 分钟
- 路线特点: 沿东三环，路况较好
- 建议: 避开早晚高峰（7:30-9:00, 17:30-19:00）

**望京 → 国贸**:
- 距离: 约 8 公里
- 预计时间: 30-35 分钟
- 路线特点: 可走机场高速辅路
- 建议: 路程较远，注意体力分配

### 休闲骑行路线
- 三里屯 → 朝阳公园: 适合周末休闲
- 国贸 → CBD 中央公园: 短途放松

## 骑行安全建议

### 安全提示
- 遵守交通规则，不闯红灯
- 夜间骑行注意照明
- 雨天路滑，减速慢行
- 佩戴头盔（推荐）

### 避开拥堵时段
- 早高峰（7:30-9:00）: 避开主干道
- 晚高峰（17:30-19:00）: 选择辅路
- 周末商圈: 三里屯、国贸等地人流密集

## 常见问题处理

### 问题1: 距离查询
**用户提问**: "从国贸到三里屯有多远？"
**处理方案**:
1. 使用 `calculate_distance` 工具计算距离
2. 告知预计骑行时间
3. 提供路线建议

### 问题2: 路线规划
**用户提问**: "我想从国贸骑到三里屯，怎么走？"
**处理方案**:
1. 使用 `plan_route` 工具规划路线
2. 提供距离和时间信息
3. 给出骑行建议和注意事项

### 问题3: 路程太远
**用户提问**: "这么远，骑车累不累？"
**处理方案**:
1. 根据距离评估难度
2. 5公里以内: 适合骑行
3. 5-10公里: 建议体力好的用户
4. 10公里以上: 推荐其他交通方式
