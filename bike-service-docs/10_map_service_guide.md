# 地图服务使用指南

## 问题类型
- **问题名**: `MapServiceGuide`
- **问题级别**: 低优先级
- **触发条件**: 用户询问地图功能、位置查询或地理相关问题

## 问题描述
用户需要了解如何使用地图服务功能：
- 搜索地点位置
- 查询两地距离
- 规划骑行路线
- 查找附近停放点

## 地图服务功能

### 功能1: 地点搜索
**工具**: `search_location`
**用途**: 搜索地点的经纬度和详细地址
**示例**: 搜索"国贸"、"三里屯"等地点

### 功能2: 距离计算
**工具**: `calculate_distance`
**用途**: 计算两个地点之间的距离
**示例**: 计算国贸到三里屯的距离

### 功能3: 路线规划
**工具**: `plan_route`
**用途**: 规划骑行路线，提供距离和时间
**示例**: 规划从国贸到三里屯的骑行路线

## 使用场景

### 场景1: 查找停放点
**用户需求**: "我在国贸，附近哪里可以停车？"
**处理流程**:
1. 使用 `search_location` 搜索国贸位置
2. 结合停放规则知识库
3. 告知最近的正规停放点

### 场景2: 距离咨询
**用户需求**: "从三里屯到望京有多远？"
**处理流程**:
1. 使用 `calculate_distance` 计算距离
2. 告知预计骑行时间
3. 提供是否适合骑行的建议

### 场景3: 路线规划
**用户需求**: "我想从国贸骑到三里屯，怎么走？"
**处理流程**:
1. 使用 `plan_route` 规划路线
2. 提供详细的距离和时间
3. 给出骑行建议和注意事项

## 注意事项

### 数据准确性
- 地图数据基于高德地图 API
- 距离为实际骑行距离，非直线距离
- 时间为预估值，实际可能有偏差

### 使用限制
- 仅支持北京地区查询
- 需要提供具体地点名称
- 模糊查询可能返回多个结果

## 常见问题

### 问题1: 找不到地点
**现象**: 搜索地点时提示"未找到该地点"
**原因**: 地点名称不准确或不在支持范围
**解决**: 使用更具体的地点名称或地标

### 问题2: 距离不准确
**现象**: 计算的距离与实际感觉不符
**原因**: 返回的是骑行路线距离，非直线距离
**说明**: 实际骑行需绕路，距离会更长
