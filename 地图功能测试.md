# 地图功能测试指南

## 测试前准备

1. 启动 Milvus 数据库
2. 启动 Spring Boot 应用
3. 索引地图文档：
   ```bash
   curl -X POST "http://localhost:9900/api/milvus/index-directory?directoryPath=bike-service-docs"
   ```

## 测试用例

### 测试1：查询停放点
**用户输入：** "我在国贸，附近哪里可以停车？"
**期望：** Agent 调用 search_location 工具搜索国贸位置，并结合知识库返回停放规则

### 测试2：计算距离
**用户输入：** "从国贸到三里屯有多远？"
**期望：** Agent 调用 calculate_distance 工具返回距离（约4.31公里）

### 测试3：路线规划
**用户输入：** "我想从国贸骑到三里屯，怎么走？"
**期望：** Agent 调用 plan_route 工具返回路线和预计时间（约15分钟）
